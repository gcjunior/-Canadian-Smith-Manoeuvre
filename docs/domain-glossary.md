# Domain Glossary

| Term                                       | Definition                                                                                                                                                                                                                         |
| ------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Smith Manoeuvre**                        | Canadian strategy converting non-deductible mortgage principal paydown into deductible investment debt via a readvanceable mortgage / HELOC structure. This product **simulates operations only** and does not provide tax advice. |
| **Tenant**                                 | Isolation root for multi-tenancy (household or firm).                                                                                                                                                                              |
| **Strategy**                               | User-configured automation binding accounts, ETF, caps, timezone, and expected payment day.                                                                                                                                        |
| **Payment period**                         | Identifier for a single expected monthly mortgage payment (e.g. `2026-07`).                                                                                                                                                        |
| **Conversion cycle**                       | One automated run converting principal paydown into a HELOC-funded ETF purchase attempt for a payment period.                                                                                                                      |
| **Interest cycle**                         | Separate run confirming HELOC interest charge and ordinary-bank debit for an interest period.                                                                                                                                      |
| **Readvanceable mortgage**                 | Simulated structure where mortgage principal reduction increases HELOC available credit.                                                                                                                                           |
| **HELOC**                                  | Home equity line of credit; revolving borrowing facility used for investment draws in this simulator.                                                                                                                              |
| **Available credit**                       | Amount that can still be drawn on the HELOC.                                                                                                                                                                                       |
| **Newly available HELOC credit**           | Increase in available credit attributable to the latest settled mortgage principal reduction (definition refined in implementation; see unresolved questions).                                                                     |
| **Principal repaid**                       | Portion of a settled mortgage payment that reduced mortgage principal (integer cents).                                                                                                                                             |
| **User monthly cap**                       | Maximum draw the user allows per conversion cycle (cents).                                                                                                                                                                         |
| **Platform monthly cap**                   | Maximum draw the platform allows per conversion cycle (cents).                                                                                                                                                                     |
| **Draw amount \(D\)**                      | `\min(principal, newlyAvailableCredit, userCap, platformCap)`.                                                                                                                                                                     |
| **Ordinary bank account / BANK_OPERATING** | Day-to-day deposit account used to pay HELOC interest.                                                                                                                                                                             |
| **Non-registered brokerage**               | Taxable investment account (simulator); destination for investment funds.                                                                                                                                                          |
| **Fractional market order**                | Notional buy of a simulated ETF allowing fractional quantities.                                                                                                                                                                    |
| **SETTLED**                                | Provider status meaning funds/position movement is final for simulator purposes.                                                                                                                                                   |
| **Idempotency key**                        | Unique key for a financial POST preventing duplicate side effects.                                                                                                                                                                 |
| **Reconciliation**                         | Verification that provider states and ledger legs form a complete balanced money trail under domain rules.                                                                                                                         |
| **Safety pause**                           | Strategy state preventing further automated draws/orders until cleared.                                                                                                                                                            |
| **Ledger entry**                           | Append-only financial fact in CAD cents for an account.                                                                                                                                                                            |
| **Audit event**                            | Append-only operational record of significant actions and outcomes (redacted).                                                                                                                                                     |
| **Correlation ID**                         | UUID linking API request, workflow, activities, provider calls, ledger, and audit.                                                                                                                                                 |
| **Deterministic scenario**                 | Fixed simulator script for tests.                                                                                                                                                                                                  |
| **Seeded failure mode**                    | Demo-only pseudo-random failures driven by a seed inside Activities/simulators.                                                                                                                                                    |
| **Temporal Schedule**                      | Durable calendar trigger starting workflows after expected payment/interest windows.                                                                                                                                               |
| **Signal**                                 | Temporal mechanism to wake a workflow (e.g. bank webhook).                                                                                                                                                                         |
| **Modular monolith**                       | Single codebase / DB with clear module boundaries and separate deployable processes.                                                                                                                                               |
| **Non-goal: capitalizing HELOC interest**  | Drawing the HELOC (or using investments) to pay HELOC interest — forbidden.                                                                                                                                                        |

## Account kinds (quick reference)

`MORTGAGE` · `HELOC` · `BANK_OPERATING` · `BROKERAGE_CASH` · `BROKERAGE_POSITION`

## Status vocabulary (quick reference)

`PENDING` · `SUBMITTED` · `SETTLED` · `FAILED` · `CANCELLED` · `TIMED_OUT_NEEDS_RECONCILE`
